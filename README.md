# GPS - Jetpack Compose

Aplicativo Android desenvolvido com **Jetpack Compose** para obter e acompanhar coordenadas de localização em tempo real (**latitude** e **longitude**) usando o serviço de localização.

## Funcionalidades

- Solicitação de permissão de localização em tempo de execução
- Início e pausa do rastreamento de localização
- Atualização contínua das coordenadas (lat/lng)
- Interface simples com botão para iniciar/parar rastreamento

## Tecnologias utilizadas

- Kotlin
- Jetpack Compose
- Material 3
- Google Play Services Location
- Android SDK (minSdk 24)

## Como executar

1. Abra o projeto no Android Studio
2. Sincronize o Gradle
3. Execute em dispositivo físico ou emulador com localização habilitada
